# Python Attendance System

This project is a simple Python-based attendance checker.

## 📌 Features
- Reads student list from CSV file
- Compares with attendance list
- Marks:
  - "+" for present
  - "-" for absent
- Outputs results to a new CSV file

## 🛠 Technologies
- Python 3
- CSV module

## ▶️ How to Run

1. Prepare `students.csv` and `attendance.csv`
2. Run the script:

```bash
python main.py
